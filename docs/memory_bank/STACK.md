# Stack Technique - MealPlanner

## Vue d'ensemble

Application fullstack avec séparation Backend/Frontend :
- **Backend** : API REST en PHP Laravel
- **Frontend** : Application web en Nuxt 3 (Vue.js)

---

## Backend

### Framework principal
- **Laravel** (PHP)
  - Framework PHP moderne pour la création d'API REST
  - ORM Eloquent pour la gestion de la base de données
  - Système de migration et de seeding
  - Validation de données intégrée
  - Système d'authentification et d'autorisation

### Base de données
- **MySQL**
  - Stockage des utilisateurs et préférences
  - Stockage des recettes et ingrédients
  - Plans de repas et listes de courses
